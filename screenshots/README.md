# Screenshots

This directory contains screenshots of the Smoothie application for the README.

## Adding Screenshots

To add new screenshots:

1. Capture screenshots of the application in various states
2. Save them as PNG files in this directory
3. Update the main README.md to reference the new images
4. Use descriptive filenames (e.g., `dashboard.png`, `profiles.png`)

## Recommended Screenshots

- **dashboard.png**: Main dashboard with profile overview and system status
- **profiles.png**: Profile creation, editing, and management interface
- **monitor-editor.png**: Visual monitor layout editor with drag-and-drop positioning
- **capture.png**: System capture workflow for detecting monitors and applications
- **settings.png**: Application preferences and configuration options
- **help.png**: Help & support page with FAQ and feedback submission

## Guidelines

- Use PNG format for better quality
- Capture at standard resolutions (1920x1080 or similar)
- Include UI elements and key features
- Add descriptive alt text in README links
